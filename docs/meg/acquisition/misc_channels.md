# Miscellaneous Channels


**12 analog inputs** for miscellaneous analog signals are available (**<span style="color:red">maximum +/- 10V</span>**).<br />
Connections are made via BNC connectors to a separate interface box similar to STI101 & STI102.

<big>
**<span style="color:maroon">Current MISC connections</span>**

1. EyeLink 1000 Plus DAC0 - **X** - horizontal position
2. EyeLink 1000 Plus DAC1 - **Y** - vertical position
3. EyeLink 1000 Plus DAC2 - **P**upil size
4. Light-2-Voltage Converter (L2V) - **#1**
5. Light-2-Voltage Converter (L2V) - **#2**
6. Stereo Audio Output - **Left** - (*from Stim PC, or DATAPixx*)<br /> - see **[MEG Audio Connections](../../meg/pdfs/MEG_Audio_Connections.pdf)**
7. Stereo Audio Output - **Right** - (*from Stim PC, or DATAPixx*)
8. Microphone Input - **Left** - (*response to Stereo Audio Left Output*)
9. Microphone Input - **Right** - (*response to Stereo Audio Right Output*)
10. Accelerometer Output - **X**
11. Accelerometer Output - **Y**
12. Accelerometer Output - **Z**

</big>
