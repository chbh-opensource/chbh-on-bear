# EyeLink Analog Output

<span style=font-size:medium>In order to ensure the EyeLink "**X**", "**Y**" and "**P**" **(pupil)** are recorded on MEG channels **MISC#1, MISC#2** and **MISC#3**... </span>

![EyeLink Analog Output setting](../../images/meg/EyeLink_Analog_Output_setting.jpg){width=45% align=left}<br />

- Make sure **MISC#1, MISC#2**, and **MISC#3** are **selected in MEG Acquisition**.
- **Select a value for Analog Output** e.g. **<span style="color:red">Raw</span>**, in **EyeLink Set Options** (*bottom left of screen*).
- **Make sure** the **EyeLink is set to RECORD in order to see the Output**.

<br /><br /><br />
If using any of the **Binocular** settings, make sure the **LEFT eye** is **ALWAYS selected**<br /> - **DAC3, DAC4, and DAC5 are not being recorded** (*no cable connection from EyeLink Host A/D card output to MEG MISC channels*).
