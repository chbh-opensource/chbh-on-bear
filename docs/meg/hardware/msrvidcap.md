# MSR Video Capture

The AD-joined PC in the Lab contains a StarTech **[High-Definition PCIe Capture Card](https://www.startech.com/en-gb/audio-video-products/pexhdcap2)** 
used to capture/record the camera output from the MSR participant monitoring system.

Only video can be recorded, no audio and no timing information/TTL triggers can be added to the MEG data when acquiring (the provided MSR camera setup isn't designed for that).

Once the data is saved to an AD Home directory (OneDrive), the video can be reviewed after the session if a note needs to be made of any atypical participant movements.

A decent video player may need to be installed. A good one to try is **VideoLAN VLC Player** (downloaded from Software Centre, or AppsAnywhere)

**Usage:**

- Log into the PC using AD username/password (no need to power off the system after use, just log out).
- Double-click the ''StreamCatcherPro'' shortcut on the Desktop.
- Change the settings/resolution/file format to suit requirements.
- Recorded files are saved in the default directory C:\StreamCatcherPro\Record.
- Exit StreamCatcherPro once finished.
- Move recorded data to AD Home directory (OneDrive).
- Log out of the PC.

Approx. 1 minute of video is ~100MB, at the default settings, so files could get quite large very quickly.

Please adjust the settings/resolution/file format as necessary, via the **[Stream Capture Pro User Guide](../../meg/pdfs/StreamCatcherPro_SUG.pdf)** 
and there is also a printed copy of the Guide on the shelf.
