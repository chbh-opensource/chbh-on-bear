# MSR Video Capture

The AD-joined PC in the Lab contains a **StarTech** **[High-Definition PCIe Capture Card](https://www.startech.com/en-gb/audio-video-products/pexhdcap2)** 
used to capture/record the camera output from the MSR participant monitoring system.

**Only video can be recorded**, no audio and no timing information/TTL triggers can be added to the MEG data when acquiring (the provided MSR camera setup isn't designed for that).

**Once the video is saved to an AD Home directory (OneDrive)**, it can be reviewed after the session if details need to be noted of any atypical participant movements.

A decent video player may need to be installed. A good one to try is **VideoLAN VLC Player** (downloaded from Software Centre, or AppsAnywhere)

!!! Note

**<span style="color:maroon">Usage:</span>**

- **Log in** to the PC using AD username/password (*no need to power off the system after use, just log out*).
- Double-click the **OBS Studio** shortcut on the Desktop.
- **Change** the settings/resolution/file format to suit requirements.
- **Recorded files** are saved in the directory **C:\OBS_Capture**.
- **Exit** OBS Studio once finished.
- **Move** recorded data to AD Home directory (OneDrive).
- **Log out** of the PC.

**<span style="color:blue">Approx. 1 minute of video (MP4) is ~20-60MB, at the default settings.</span>**

Printed copies of the above Guides are in a ring binder on the shelf above the PC.
