# Key Safe Instructions


![Key Safe Buttons](../../images/meg/KeySafeButtons.jpg){width=39% align=left}

- ***<span style=font-size:medium;color:blue>Each number entry is confirmed by a red LED blink signal.<br />A valid code is followed by a red light signal, an invalid code by a triple blink signal.<br />After an interval of more than 10 seconds, the already entered part of the code is deleted and the entire sequence has to be re-entered.</span>***

**<span style=font-size:medium>To open the lock:</span>**

- ***<span style=font-size:medium;color:blue>Enter the code and (after a red LED light signal) the lock is open, turn the grip clockwise.<br />If the lock is not opened within 2 seconds, it will be locked automatically.</span>***

**<span style=font-size:medium>To close the lock:</span>**

- ***<span style=font-size:medium;color:blue>Close the door and turn grip counter-clockwise. The lock will close automatically.<br />Check if it is really closed by turning the grip.</span>***

**<span style=font-size:medium>Lighting the entry-pad:</span>**

- ***<span style=font-size:medium;color:blue>Press number 9 first and the entry-pad will be lit. After that, the 6-digit code will still need to be entered.<br />The LED stays on during code entry.</span>***

<br />
**<span style=font-size:x-large>Wrong code penalty:</span>**

***<span style=font-size:large;color:red>After 4 invalid codes, the entry is blocked for about 5 minutes.<br /> During this period the lock signals every 10 seconds.<br /> If, after 5 minutes, 2 wrong codes are entered, the 5-minute time delay starts again.</span>***


**<span style=font-size:large;color:maroon>Battery change:</span>**

**<span style=font-size:large;color:green>When the batteries are running low, the lock will signal/blink for about 5 seconds (3 blinks) while opening/after code entry, signifying the batteries will need replacing. Use ALKALINE batteries only! <br /> The battery compartment is behind the lock, on the inside of the door. Open the battery clip carefully.<br /> If the battery voltage is so low and insufficient to open the safe on code entry, hold a new 9V ALKALINE battery to the outside terminals and try entering the code again. The red LED should blink after successful code entry and the grip can be turned as normal. <br /> Replace the battery.</span>**
