# MaxShield state unknown

### **<span style="color:maroon">Problem</span>**

On 16th July 2018, whilst in the Tools section of the Acquisition program, the following **error message** was generated.

**```403 MaxShield state unknown - cannot measure.```**<br />
**```Turn MaxShield on or off```**

### **<span style="color:maroon">Solution</span>**

* **Restart/Reset** the DSP electronics, see **[Resetting the electronics](../../meg/acquisition/restart_acquisition.md/#resetting-the-electronics)**.
* **Perform** a **[RAP](../../meg/acquisition/rap.md)** (**R**estart **A**cquisition **P**rograms).
* **Load** the latest Tuning file, see **[Loading a Tuning File](../../meg/acquisition/restart_acquisition.md/#loading-a-tuning-file)**.

See **[Restart Acquisition](../../meg/acquisition/restart_acquisition.md)** for full details.

* If the error message is still not cleared, email MEGIN Support **(<span style="color: red">support [at] megin.com</span>)**
